ORTC over RTC
=============

A shim to implement `ORTC` over [WebRTC API](http://www.w3.org/TR/webrtc) along with test cases.

Not all `ORTC` features are supported due to missing `WebRTC API` as specified for browser.

Requirements
------------

  * Google Chrome
  * NodeJS

Use-Cases
---------

    npm install
    npm start
    open http://localhost:8081/
